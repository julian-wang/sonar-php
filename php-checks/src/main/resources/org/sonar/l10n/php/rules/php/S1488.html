声明一个变量，只是立即返回［或抛出］是一种坏的实践。
一些开发人员认为，这种做法提高了代码的可读性，因为这允许显式命名所返回的值。然而，这个变量作为内部实现的细节，是不会暴露给函数的调用者。对调用者而言，函数名应该足够明确什么会被返回。

不规范的代码示例

function compute_duration_in_milliseconds() {
  $duration = ((($hours * 60) + $minutes) * 60 + $seconds ) * 1000 ;
  return $duration;
}

规范的解决方案

function compute_duration_in_milliseconds() {
  return ((($hours * 60) + $minutes) * 60 + $seconds ) * 1000;
}
