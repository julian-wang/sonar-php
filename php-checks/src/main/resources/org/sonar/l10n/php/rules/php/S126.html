这条规则适用于 if语句，无论其后跟着一个或多个else if语句；最后一个else if 后应该跟着一个else语句。

最后必须else语句是一种防御性编程。else语句一个要么采取适当的错误，要么包含适当的注释解释为什么没有采取措施。这和switch语句中必须在最后有一个default子句是相一致的。

不规范的代码示例

if (condition1) {
  do_something();
} else if (condition2) {
  do_something_else();
}

规范的解决方案

if (condition1) {
  do_something();
} else if (condition2) {
  do_something_else();
} else {
  throw new InvalidArgumentException('message');
}
