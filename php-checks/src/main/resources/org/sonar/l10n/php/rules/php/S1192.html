重复的字符串使重构的过程非常容易出错，因为你必须更新所有字面值出现的地方。

另一方面，常量可以被多个地方引用，但是只需要在一个地方更新。

不规范的代码示例

function run() {
  prepare('action1');          // Non-Compliant - 'action1' is duplicated 3 times
  execute('action1');
  release('action1');
}

规范的解决方案

ACTION_1 = 'action1';

function run() {
  prepare(ACTION_1);
  execute(ACTION_1);
  release(ACTION_1);
}

（为了防止产生一些误判，排除了长度小于5个字符的字符串。）
