在一行块，例如 if 语句或者 for循环中，大括号可以省略，但这样做会产生误导和引入bug。

当一个一行块之后的多行代码缩进，暗示了这些行代码属于这个块时，这条规则会触发。但是花括号的省略意味着这些行会无条件执行一次。

不规范的代码示例

if ($condition)
  firstActionInBlock();
  secondAction();  // Noncompliant; executed unconditionally
thirdAction();

$str = null;
for ($i = 0; $i < count($array); $i++)
  $str = $array[$i];
  doTheThing($str);  // Noncompliant; executed only on last array element

规范的解决方案

if ($condition) {
  firstActionInBlock();
  secondAction();
}
thirdAction();

$str = null;
for ($i = 0; $i < count($array); $i++) {
  $str = $array[$i];
  doTheThing($str);
}
