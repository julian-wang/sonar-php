没有使用的参数会产生误导。无论传递给这个参数什么值，函数的行为都是相同的。

不规范的代码示例

function doSomething($a, $b) { // "$a" is unused
  return compute($b);
}

规范的解决方案

function doSomething($b) {
  return compute($b);
}

例外

重写类或者实现接口的函数会被忽略。

class C extends B {

  function doSomething($a, $b) {     // no issue reported on $b
    compute($a);
  }

}
