共享一些命名规则是使团队高效协作的一个关键，这条规则允许检查所有的函数名称是否匹配一个正则表达式。

不规范的代码示例

With default provided regular expression: ^[a-z][a-zA-Z0-9]*$

function DoSomething(){...}

规范的解决方案

function doSomething(){...}

例外

（带有@inheritdoc标注的函数, 还有魔术函数 (__construct(), __destruct(), __call(), __callStatic(), __get(), __set(), __isset(), __unset(), __sleep(), __wakeup(), __toString(), __invoke(), __set_state(), __clone(), __debugInfo()) 忽略这条规则。）

function __construct(){...}
function __destruct(){...}

/**
 * {@inheritdoc}
 */
function myFunc(){...}
