$this是指当前类的实例. 但是静态方法可以无需实例化类访问, $this是对它们是不可访问的. 在静态上下文中使用 $this会导致一个运行时错误。

不规范的代码示例

class Clazz {
  $name=NULL;  // instance variable

  public static function foo(){
    if ($this->name != NULL) {
      // ...
    }
  }
}

规范的解决方案

class Clazz {
  $name=NULL;  // instance variable

  public static function foo($nameParam){
    if ($nameParam != NULL) {
      // ...
    }
  }
}
