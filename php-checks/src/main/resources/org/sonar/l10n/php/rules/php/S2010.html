PHP有两套逻辑运算符： & / ||, 和 and / or. 两套之间的差别就是优先级。. 因为 and / or 的优先级比其他操作符优先级低, 使用它们而不是&& / ||可能导致的不到你预期的结果。

不规范的代码示例

$have_time = true;
$have_money = false;
$take_vacation = $have_time and $have_money;  // Noncompliant. $take_vacation == true.

规范的解决方案

$have_time = true;
$have_money = false;
$take_vacation = $have_time && $have_money;  // $take_vacation == false.
