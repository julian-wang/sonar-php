共享的编码规范可以使团队高效地协作。这条规则强制要求开括号必须在新的一行的开始。

下面的代码片段说明了这条规则：

function myMethod()
{                                               //Compliant
  if(something)
  {                                             //Compliant
    executeTask();
  } else {                                    //Non-Compliant
    doSomethingElse();
  }
}
