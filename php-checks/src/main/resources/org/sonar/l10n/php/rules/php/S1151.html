switch语句应该仅用于定义一些新的控制流分支。一旦case子句包含了过多的语句，就会显著降低全局控制流的可读性。在这种情况下，case子句的内容应该提取到一个专门的函数中。

不规范的代码示例

下面的代码片段展示了默认阈值为5的一个规则：

switch ($var) {
  case 0:  // 6 lines till next case
    methodCall1();
    methodCall2();
    methodCall3();
    methodCall4();
    break;
  default:
    break;
}

规范的解决方案

switch ($var) {
  case 0:
    doSomething();
    break;
  default:
    break;
}

function doSomething(){
  methodCall1("");
  methodCall2("");
  methodCall3("");
  methodCall4("");
}
