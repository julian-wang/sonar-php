没有理由重新分配一个变量给其本身。要么该语句是冗余的，应该被删除，或重新分配是一个错误，其他值或变量的才是应该赋值的。

不规范的代码示例

public function setName($name) {
    $name = $name;
}

规范的解决方案

public function setName($name) {
    $this->name = $name;
}
