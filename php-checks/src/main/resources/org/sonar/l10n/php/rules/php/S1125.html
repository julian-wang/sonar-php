从条件表达式中删除文本布尔值，以提高可读性。任何可以与布尔值测试相等的值，本身必须是一个布尔值，所以可以自动的测试其布尔值。

不规范的代码示例

if ($booleanVariable == true) { /* ... */ }
if ($booleanVariable != true) { /* ... */ }
if ($booleanVariable || false) { /* ... */ }
doSomething(!false);

$booleanVariable = condition ? true : exp;
$booleanVariable = condition ? false : exp;
$booleanVariable = condition ?  exp : true;
$booleanVariable = condition ?  exp : false;

规范的解决方案

if ($booleanVariable) { /* ... */ }
if (!$booleanVariable) { /* ... */ }
if ($booleanVariable) { /* ... */ }
doSomething(true);

$booleanVariable = condition || exp;
$booleanVariable = !condition && exp;
$booleanVariable = !condition ||  exp;
$booleanVariable = condition && exp;

例外

在同一性操作符(=== 和 !==)中使用布尔值字面值忽略此规则.
