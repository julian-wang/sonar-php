全局变量是一个有用的结构，但是它们不应该被滥用．函数应该通过关键字global 或者通过 $GLOBALS 数组访问全局作用域, 但这种做法大大降低了函数的可读性和可重用性. 相反的, 全局变量应该作为一个参数传给函数．

不规范的代码示例

$myGlobalVariable;

function foo()
{
  global $myGlobalVariable;     // Noncompliant
  $GLOBALS['myGlobalVariable']; // Noncompliant
  // ...
}

规范的解决方案

function foo($myStateVariable)
{
  // ...
}
