因为很容易从编译的应用程序中提取字符串，所以凭证密钥不应该硬编码。这样做，几乎可以确定它会落到攻击者手里，特别是对分布式应用来说。

凭证密钥应该储存在代码之外的、强保护的加密配置文件中或数据库中。

不规范的代码示例

$uname = "steve";
$password = "blue";
connect($uname, $password);

规范的解决方案

$uname = getEncryptedUser();
$password = getEncryptedPass();
connect($uname, $password);
