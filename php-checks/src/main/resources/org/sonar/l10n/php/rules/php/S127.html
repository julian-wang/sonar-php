for 循环的停止条件必须是不变的（即每个循环迭代的开始和结束都应该为true）。理想条件下，这意味这在循环开始之前，停止条件被设置为局部变量。

停止条件可变会难以理解和维护，这会导致错误在未来被引入。

此规则跟踪三种类型的非不变的停止条件：

  • 当循环计数器在 for循环中更新时
  • 当停止条件依赖函数调用时
  • 当停止条件取决于对象属性，而这种属性在循环执行期间可以改变时。

不规范的代码示例

for ($i = 0; $i < 10; $i++) {
  echo $i;
  if(condition) {
    $i = 20;
  }
}

规范的解决方案

for ($i = 0; $i < 10; $i++) {
  echo $i;
}

参见

  • MISRA C:2004, 13.6 - Numeric variables being used within a for loop for iteration counting shall not be modified in the body of the loop.
  • MISRA C++:2008, 6-5-3 - The loop-counter shall not be modified within condition or statement.
