PHP是一种强大的语言，允许为函数参数定义一个默认值．但是所有带有默认值的函数参数都应该放在不带默认值的函数参数之后．

不规范的代码示例

在下面的例子中，调用makeyogurt("raspberry") 并不会得到预期的结果：

function makeyogurt($type = "acidophilus", $flavour){...}

下面的警告会在运行时产生： Missing argument 2 in call to makeyogurt()

规范的解决方案

function makeyogurt($flavour, $type = "acidophilus", ){...}
