goto是一种非结构化的控制流语句。它降低了代码的可读性和可维护性。结构化的控制流语句，例如 if, for, while, continue or break 应该用来替代goto。

不规范的代码示例

$i = 0;
loop:
  echo("i = $i");
  $i++;
  if ($i < 10){
    goto loop;
  }

规范的解决方案

for ($i = 0; $i < 10; $i++){
  echo("i = $i");
}
