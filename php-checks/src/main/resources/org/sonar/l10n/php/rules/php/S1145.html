if(false)语句使代码块没有功能，if(true)语句纯粹是多余，而且使代码可读性变差。

这里有三种出现这种代码的可能：

  • 调试的时候修改了if语句，而且调试代码被提交了。
  • 有些变量没有被重置。
  • 有些逻辑没有按照程序员认为的那样执行。

在这些条件下，没有条件的if 语句都应该被移除。

不规范的代码示例

if (true) {  // Noncompliant
  doSomething();
}
...
if (false) {  // Noncompliant
  doSomethingElse();
}

规范的解决方案

doSomething();
