函数中包含过多的return语句，增加了函数实质的复杂性，因为每次遇到返回语句，都会打断运行流程。这使得它更难读和理解的逻辑功能。

不规范的代码示例

下面的代码片段显示了一个阈值为3的规则： 

function myFunction(){ // Non-Compliant as there are 4 return statements
  if (condition1) {
    return true;
  } else {
    if (condition2) {
      return false;
    } else {
      return true;
    }
  }
  return false;
}
